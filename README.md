# 📊 Suicide Trend Analysis

**Suicide Trend Analysis** is a data analytics project developed using **R** and **Shiny** to explore suicide trends, demographic patterns, and risk factors across India. The project combines data preprocessing, exploratory data analysis (EDA), statistical hypothesis testing, and interactive visualizations to generate meaningful insights from publicly available suicide datasets.

---

## 📌 Project Overview

The primary objective of this project is to analyze suicide data across different states, age groups, genders, marital statuses, and causes to identify significant trends and relationships. Using R for statistical analysis and Shiny for dashboard development, the project transforms raw data into interactive visualizations that support data-driven decision-making.

---

## 📈 Statistical Analysis

The project applies **Chi-Square statistical tests** to examine relationships between key variables.

### Hypotheses Tested

* Gender vs. Suicide Cause
* Marital Status vs. Gender
* State-wise Suicide Distribution

The results showed statistically significant relationships (**p < 0.05**) across all tested hypotheses, indicating that demographic and geographic factors have a significant association with suicide patterns.
